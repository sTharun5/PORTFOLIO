# Tharun S - Backend Developer Portfolio

A minimalist, high-performance developer portfolio built to showcase backend engineering expertise.

## Tech Stack
- **Frontend Framework**: Vue 3 (Composition API)
- **Styling**: Tailwind CSS v4
- **Animations**: @vueuse/motion
- **Validation**: TypeScript
- **Build Tool**: Vite

## Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/sTharun5/PORTFOLIO
   cd PORTFOLIO
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Design Philosophy
This project shuns heavy grids and generic templates in favor of a specialized, human-designed split-pane layout. It is extremely lightweight, prioritizes whitespace and typography, and is geared specifically toward emphasizing server-side architecture and logic.

## Deployment
This architecture is completely static and pre-compiled, configured for zero-downtime deployment on platforms like Netlify. Committing to the `main` branch inherently triggers a strict production build.
